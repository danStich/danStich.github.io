Introduction to R for Aquatic Research

2018 NALMS International Symposium

Welcome to the home page for the Introduction to R for Aquatic Research Workshop at the 2018 International NALMS Symposium.

This workshop will consist of two sessions. The morning session (8:00 am - 12:00 pm) will be an introduction to R as a functional, vectorized, and object-oriented statistical programming language. It will cover R syntax (e.g., how to read and write R code), types of variables and objects, functions, data management (e.g., loading, referencing, manipulation, transformation, and subsets), basic descriptive statistics, and working with R packages. The afternoon session (1:00 - 5:00 pm) will build on these techniques using the Secchi Dip-In data and other data sets for an Intermediate R Workshop. Topics will include regression analyses, predictions, simulation, and data visualization. Those less familiar with R should plan on attending both sessions, but those who have some experience using R could consider skipping the morning session and join during lunch for a half-day session. Laptop computers are required for both sessions. All software is freely available and needs to be installed prior to the workshop because we will be working without internet .
